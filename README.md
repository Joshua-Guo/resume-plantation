# 农场主.skill

> 农场主放下手里的账本，低头看着眼前这份简历，沉默了三秒。
>
> **"小简历，真得好好调教你了。"**

---

## 这玩意儿是干什么的

你投了 80 份简历，收到 3 封感谢信，1 个 HR 已读不回。
你觉得是市场的问题。
但其实——

```
"负责后端服务开发，参与系统优化，提升了系统性能。"
```

这句话里有 4 个动词：负责、参与、优化、提升。
含金量：0。

**简历农场主**是一个给 Claude Code 用的 skill，专门把"职责说明书"改造成"成就展示板"。农场主不说废话，直接调教。

---

## 它怎么用

农场主从头到尾帮你 debug 简历，六步走：

| 步骤 | 干什么 |
|------|--------|
| **第零步：农场主评分** | 总分 + 头衔 + 分项加权表，先让你知道自己几斤几两 |
| 验货 | 5秒第一判决，能打还是寄了 |
| 下地干活 | 逐行锐评，每个问题三连（是什么→怎么死的→怎么救）|
| 调教方案 | 成果叙事公式 + 挖出你自己都忘了的亮点 |
| 改造后出栏 | 完整重写版简历，Markdown 格式，直接复制 |
| 最终验收 | 高危项复盘 + 出院任务单 |

#### 农场主评分体系

农场主先看人，再开口。九档头衔，按总分定级：

| 分数 | 头衔 | 农场主旁白 |
|------|------|-----------|
| 0 - 19 | 🪰 纯种垃圾 | 连做肥料都嫌重金属超标 |
| 20 - 34 | 🦴 待焚废料 | 活着就是对空气的亵渎 |
| 35 - 49 | 🐕 下等贱畜 | 仅存的价值是那身皮肉 |
| 50 - 62 | ⛓️ 耐操耗材 | 只要不打死，就能一直干 |
| 63 - 74 | 🚜 人形农具 | 莫得感情的干活机器 |
| 75 - 84 | 🐎 极品骡马 | 肌肉发达，值得加点草料 |
| 85 - 92 | 💰 纯血黑金 | 拍卖场上的顶级尖货 |
| 93 - 99 | 👔 黑皮管家 | 替主子挥鞭的忠诚走狗 |
| 100 | 🍷 荣誉白皮 | 准许进屋伺候的二主子 |

总分由六个维度加权计算：

| 维度 | 权重 | 为什么这个权重 |
|------|------|--------------|
| 工作经历描述质量 | 30% | 决定能不能过简历筛选 |
| 项目含金量 | 25% | 区分度最高，拉开差距全靠这里 |
| 职业故事线 | 15% | 影响 HR 的第一判断 |
| 个人摘要 | 10% | 10秒能不能抓住眼球 |
| 技能匹配度 | 10% | 关键词有没有对上 JD |
| AI 前瞻性 | 10% | 2026年了，懂不懂用 AI |

```
总分：52 / 100
```

| 维度 | 满分 | 权重 | 得分 | 加权得分 | 一句话判断 |
|------|------|------|------|---------|-----------|
| 个人摘要 | 10 | 10% | 0 | 0.0 | 完全缺失，农场主皱眉 |
| 职业故事线 | 10 | 15% | 5 | 0.75 | 10个月空白期没解释，三个月就跑路 |

### 锐评示范

**整治前：**
```
负责后端服务开发，参与系统优化，提升了系统性能。
```

**农场主锐评：**

> 🌾 **问题在哪**：全是被动动词，"负责""参与"是最没含金量的两个字。这句话翻译过来是：我在场，但我不确定我做了什么。
>
> 💀 **怎么死的**：面试官看到这种描述，脑子里自动跳过。你不是唯一一个"参与系统优化"的人，你只是最模糊的那个。
>
> 🩹 **怎么救**：`为解决[具体问题]，我[做了什么]，将[指标]从[X]优化至[Y]，[业务影响]`

**整治后：**
```
识别到主搜索接口 P99 延迟超 800ms 导致转化率下滑，
引入查询缓存层并重构索引策略，P99 降至 120ms，
对应页面转化率提升 12%。
```

同一件事，含金量拉满。

## 适合哪些人

- 📮 投了几十家没有回音，想知道简历哪里出了问题
- 🔄 准备跳槽，需要把"干了什么"改造成"成就了什么"
- 💀 就是想让 AI 不留情面地整治一下自己

---

## 它不会做什么

- ❌ 不帮你造假。一个字都不编，占位符标注你自己填
- ❌ 不只是阴阳。每个问题都有解法，破防完了给糖吃
- ❌ 不说废话。"建议加强这方面的表达"这种话，农场主说不出口
